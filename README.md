# מטרת הפרויקט
```text
לבנות אפליקצייה שמציגה למהנדס המירוץ snapeshort של תור ב Pit stop שקורה במהלך המירוץ כמה פעמים כדי להחליף צמיגים ותדלוק , המהנדס יוכל לראות תמונת מצב של התור ל Pit Stop שזה כולל מי כבר נכנס לתור בהקפה הנוכחית ומי סיים מי מחכה ומי הבר בתור וגם מאפשרת למהנדס לשלוח הודעת רדיו שזבה אומר הדפסה לנהג הבא שהוא יכנס
```
# הרצת הפרויקט
**איזה קובץ מריצים**
``` text
הפרויקט ירוץ דרך קובץ Main.js שהוא ידפיס לטרמינל את כל התמונת מצב של התור 
```
**איך מריצים**
``` bash
node main.js
```
# מבנה הקבצים
**תיקיית data**
**קבצים**
```text
data.json - שומר את הנתונים שקיבלנו על המירוץ מהכתובת API ומאחסן אותם בקובץ json
```
**תיקיית service**
**קבצים**
```text
file_service.js - אחראי לקרוא את הנתונים מהשרת API , ואחראי לכתוב את אותם נתונים בקובץ data.jason לאיחסון הנתונים
manager_service.js - פונקציות הליבה שאחראיות לקבל מצב מלא של הpit stop 
```
**קבצים נפרדים**
```text
.gitignore - של יעלה לgit את ההורדות של הספריות node_modules
main.js - הקובץ הראשי שמריץ ומדפיס לטרמינל
package.json -     החבילה שכוללת בתוכה מידע על type הפרויקט ירוץ ועוד מידעאיזה 
README.md - מתאר את מטרת ומבנה הפרויקט
```

# רשימה מה האפליקצייה מדגימה
 לוח מצב מלא של Pit stop שזה כולל : שם מירוץ באיזה הקפה נמצאים כמות מספר הרכבים כמה pit stop הסתיימו ורשימה של כמה ממתינים בתור וכן תוצאה של חיפוש רכב קיים או לא קיים  עם הודעת סיום בסוף.
